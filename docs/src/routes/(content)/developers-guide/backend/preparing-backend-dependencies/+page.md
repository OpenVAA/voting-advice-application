# Backend Setup

> **Migrated to Supabase.** The Strapi backend was removed in v3.0. This page previously documented preparing Strapi backend dependencies.

## Supabase Equivalent

No separate backend dependency installation is needed. Run `supabase start` in the `apps/supabase/` directory to launch all backend services locally. The Supabase CLI handles all dependencies automatically.
