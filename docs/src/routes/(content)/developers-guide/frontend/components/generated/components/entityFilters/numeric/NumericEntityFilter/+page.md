# NumericEntityFilter

Render a numeric filter for entities.

### Properties

- `filter`: The filter object
- `targets`: An array of target entities or rankings
- Any valid attributes of a `<div>` element

### Usage

```tsx
<NumericEntityFilter {filter} targets={candidates}/>
```
