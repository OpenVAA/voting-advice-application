# Initial data

Before publishing the candidate application, you will need to the data below. The developer you’re working with will probably provide you spreadsheet templates to fill.

- [ ] Details of the election or elections
- [ ] Details of constituencies for each election
- [ ] Statement categories
- [ ] Statements
- [ ] Statements’ supporting information
- [ ] Non-statement questions
- [ ] Candidate lists if using the candidate app and not using bank authentication
- [ ] Information content about the elections
- [ ] Privacy statements
- [ ] Any other customized text content
- [ ] Assets for visual customization
