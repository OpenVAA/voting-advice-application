<interfaces>
<!-- Playwright tag property syntax (available since 1.42, project uses 1.58.2) -->

From @playwright/test:
```typescript
// Tag on test.describe() — all tests inside inherit the tag
test.describe('name', { tag: ['@voter', '@smoke'] }, () => { ... });

// Tag on individual test (NOT recommended for this project — use describe-level)
test('name', { tag: '@smoke' }, async ({ page }) => { ... });
```

Filtering via CLI:
```bash
playwright test --grep @smoke      # Run only smoke-tagged
playwright test --grep @voter      # Run only voter-tagged
playwright test --grep @candidate  # Run only candidate-tagged
playwright test --grep @variant    # Run only variant-tagged
```

Key behavior: Tags inherit from describe to contained tests. Setup/teardown projects use `setup()` not `test()`, so `--grep` filtering does not affect them — project dependencies still run.
</interfaces>
