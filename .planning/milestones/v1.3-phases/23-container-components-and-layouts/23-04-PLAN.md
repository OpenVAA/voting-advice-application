    - Add `<svelte:options runes />`
    - Add `import type { Snippet } from 'svelte'`

      ```typescript
      let {
        title,
        titleClass = undefined,
        primaryActionsLabel = undefined,
        note,
        hero,

        fullWidth,

        children,
      ```

    - `$$restProps` -> `...restProps`

    **Slot-to-snippet conversions (6 named + 1 default):**

    - `$$slots.note` conditional -> `note` truthiness: `{#if note}`
    - `<slot name="note" />` -> `{@render note()}`
    - `$$slots.hero` conditional -> `hero` truthiness: `{#if hero}`
    - `<slot name="hero" />` -> `{@render hero()}`
    - `<slot name="heading">` (with default content inside) -> check if heading snippet is provided:
      ```svelte
      {#if heading}
        {@render heading()}
      {:else}
        <h1 class={titleClass ?? ''}>{title}</h1>
      {/if}
      ```
      This replaces the pattern `<slot name="heading"><h1 ...>{title}</h1></slot>` which used the default slot content as fallback.
    - `<slot />` (default) -> `{@render children?.()}`
    - `$$slots.fullWidth` conditional -> `fullWidth` truthiness
    - `<slot name="fullWidth" />` -> `{@render fullWidth()}`
    - `$$slots.primaryActions` conditional -> `primaryActions` truthiness
    - `<slot name="primaryActions" />` -> `{@render primaryActions()}`

    **MainContent.type.ts update:**
    - Add snippet props to MainContentProps:
      ```typescript
      import type { Snippet } from 'svelte';
      // ... existing imports ...

      export type MainContentProps = SvelteHTMLElements['div'] & {
        title: string;
        titleClass?: string;
        primaryActionsLabel?: string;
        note?: Snippet;
        hero?: Snippet;
        heading?: Snippet;
        fullWidth?: Snippet;
        primaryActions?: Snippet;
        children?: Snippet;
      };
      ```
    - Read the existing type file first to preserve all existing properties and only add the snippet props.

    IMPORTANT: Verify what `$$slots` checks exist in MainContent.svelte. The file uses `$$slots.note`, `$$slots.hero`, `$$slots.fullWidth`, `$$slots.primaryActions` as conditional wrappers. Each of these becomes a simple truthiness check on the corresponding snippet prop.

    The `heading` slot has default content (fallback `<h1>` when no heading is provided). In Svelte 5, this becomes an explicit `{#if heading}...{:else}...{/if}` block.
