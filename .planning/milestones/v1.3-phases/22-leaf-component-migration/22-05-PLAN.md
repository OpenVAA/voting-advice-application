**AppLogo.svelte (COMP-08 - svelte:component):**
Replace `<svelte:component this={LogoComponent} {...logoProps} />` with `<LogoComponent {...logoProps} />`:
```svelte
<!-- BEFORE -->
<svelte:component this={LogoComponent} {...logoProps} />

<!-- AFTER -->
<LogoComponent {...logoProps} />
```
In Svelte 5, components are first-class values -- you can use them directly without `svelte:component`. The `LogoComponent` variable (likely assigned from a dynamic import or conditional) can be used as a regular component tag since capitalized tags are treated as component references.
