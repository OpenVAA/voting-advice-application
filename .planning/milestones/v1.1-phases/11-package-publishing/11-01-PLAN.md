This replaces the old build pipeline with the modern two-step pattern: tsup (fast esbuild-powered ESM output) + tsc --emitDeclarationOnly (reliable .d.ts generation for monorepo project references).

Purpose: Unified build tooling across all packages, eliminates tsc-esm-fix dependency, produces correct ESM output without post-processing.
Output: All packages building via tsup with output in dist/, unit tests passing.
