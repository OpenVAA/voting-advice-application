## Critical Anti-Patterns (do NOT repeat these)

| Pattern | Description | Severity | Prevention |
|---------|-------------|----------|------------|
| Single-RPC array drop | Reading `options.electionId[0]` on an array param silently loses every other element | advisory | Fan out into one RPC per element when the array has multi values; deduplicate by primary key |
| `const` capturing dataRoot at module init | Const-bound `$dataRoot.getConstituencyGroup(id)` runs once before the root-layout `$effect` populates the dataRoot, freezing the page | advisory | Use `$derived` for any expression that reads through the bridged store |
| Throwing data-model getters in `$derived` | A throw inside a `$derived.by` surfaces as a page error and blocks rendering | advisory | Wrap throw-prone getters in `try/catch` and pick a deterministic fallback (e.g. `find()` first match) |
| Parallel-test app_settings drift | Tests calling `client.updateAppSettings(...)` against a shared Supabase row in parallel runs interleave and break unrelated specs | advisory | Add `test.beforeEach` that resets the keys you depend on |
