# Plan 38-01: Delete Strapi workspace and workspace entries

<objective>
Remove the entire apps/strapi/ directory and all root-level references to it — workspace entries, scripts, dependabot config — so the monorepo no longer contains any Strapi workspace.
</objective>

## Tasks

### Task 1: Delete apps/strapi/ directory

<read_first>
- `apps/strapi/package.json` (confirm workspace name is `@openvaa/strapi`)
</read_first>

<action>
Delete the entire `apps/strapi/` directory recursively.

```bash
rm -rf apps/strapi/
```
</action>

<acceptance_criteria>
- `test ! -d apps/strapi/` exits 0 (directory gone)
- `find . -path './apps/strapi' -type d 2>/dev/null | wc -l` returns 0
</acceptance_criteria>

### Task 2: Remove Strapi workspace entries from root package.json

<read_first>
- `package.json` (root)
</read_first>

<action>
In the root `package.json`:

1. Remove `"apps/strapi/src/plugins/*"` from the `workspaces` array. The resulting workspaces should be:
```json
"workspaces": [
  "packages/*",
  "apps/*"
]
```

2. Remove the `sync:translations` script that references apps/strapi:
```json
"sync:translations": "rsync -av --include='*/' --include='dynamic.json' --exclude='*' apps/frontend/src/lib/i18n/translations/ apps/strapi/src/util/translations/"
```
</action>

<acceptance_criteria>
- `grep -c 'strapi' package.json` returns 0
- `grep -c 'plugins' package.json` returns 0 (the strapi plugins workspace line is gone)
- `grep -c 'sync:translations' package.json` returns 0
- `node -e "const p=require('./package.json'); console.log(JSON.stringify(p.workspaces))"` outputs `["packages/*","apps/*"]`
</acceptance_criteria>

### Task 3: Remove Strapi entry from dependabot.yml

<read_first>
- `.github/dependabot.yml`
</read_first>

<action>
Remove the entire update entry block for `directory: "/apps/strapi"` from `.github/dependabot.yml`. The file has multiple `- package-ecosystem: npm` entries — remove only the one with `directory: "/apps/strapi"`.
</action>

<acceptance_criteria>
- `grep -c 'strapi' .github/dependabot.yml` returns 0
- `.github/dependabot.yml` is valid YAML (other entries intact)
</acceptance_criteria>

### Task 4: Remove Strapi data provider test directory

<read_first>
- `apps/frontend/tests/strapiDataProvider/strapiDataProvider.test.ts` (confirm this is Strapi-specific)
</read_first>

<action>
Delete the entire `apps/frontend/tests/strapiDataProvider/` directory:

```bash
rm -rf apps/frontend/tests/strapiDataProvider/
```
</action>

<acceptance_criteria>
- `test ! -d apps/frontend/tests/strapiDataProvider/` exits 0
</acceptance_criteria>
