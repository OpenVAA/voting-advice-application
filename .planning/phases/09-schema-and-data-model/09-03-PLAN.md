<objective>
Extend @openvaa/data with a QuestionTemplate class that defines default properties, answer type, and configuration for creating questions (renamed from Strapi's questionType concept).

Purpose: QuestionTemplate allows shared question definitions (e.g., a "5-point Likert" template whose choices are reused across many opinion questions). This is a first-class DataObject in the data model, corresponding to the question_templates database table.
Output: QuestionTemplate type, class, and tests with full DataRoot integration.
