Update the DataWriter interface, abstract base class, both adapter implementations, candidateUserDataStore, and register route pages to remove checkRegistrationKey, simplify register signature, change write method return types from LocalizedCandidateData to narrower types, and remove image from EditableEntityProps.

         const saved = get(savedData);
         if (!saved) throw new Error('Save called before user data loaded');

         const termsOfUseAccepted = get(editedTermsOfUseAccepted);

           if (!updatedAnswers) throw new Error('Failed to update answers');
           // Merge returned answers into saved candidate data
           savedData.update((s) => {
             if (!s) throw new Error('Cannot update candidate data before loaded');
             return {
