```json
}
```

Existing externalId conventions:
- Elections: "test-election-1"
- Constituencies: "test-constituency-alpha"
- Constituency Groups: "test-cg-1"
- Question Types: "test-qt-likert5"
- Questions: "test-q-{N}" (default), "test-voter-q-{N}" (voter)
- Candidates: "test-candidate-alpha", "test-voter-cand-{N}"
- Parties: "test-party-a", "test-party-b"

Import order (critical for FK resolution):
questionTypes -> constituencies -> constituencyGroups -> elections -> questionCategories -> questions -> parties -> candidates -> alliances -> nominations

Delete order (reverse of import):
nominations -> alliances -> parties -> questions -> questionCategories -> constituencyGroups -> constituencies -> elections -> questionTypes
