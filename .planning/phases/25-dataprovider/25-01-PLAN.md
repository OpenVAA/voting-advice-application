    - Import `getLocalized` from `./getLocalized`
    - Export function `localizeRow(row: Record<string, unknown>, fields: string[], locale: string, defaultLocale: string = 'en'): Record<string, unknown>`
    - Shallow-clone the row with `{ ...row }`
    - For each field in `fields`:
      - If field contains '.': split by '.', traverse the cloned object, shallow-clone each intermediate object along the path.     - Return the result object.
