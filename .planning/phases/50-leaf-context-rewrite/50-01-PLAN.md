<objective>
Rewrite I18nContext from Readable store wrapping to plain values, update the type definition, and update all consumer components that use $locale/$locales to use direct property access.

Purpose: I18nContext is the simplest leaf context -- locale and locales are constant within a page lifecycle (locale changes trigger full page reloads). Removing the Readable wrapper simplifies the API and eliminates unnecessary store overhead. This change propagates through ComponentContext and AppContext types automatically.

Output: Rewritten i18nContext.ts and i18nContext.type.ts, updated dataContext.ts and candidateUserDataStore.ts, updated 8 consumer components with direct property access.
</objective>
