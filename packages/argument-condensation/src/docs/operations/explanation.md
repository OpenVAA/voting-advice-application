## REFINE

Takes in comments batches and progressively builds/refines a single argument list by processing said batches sequentially. A separate initialization prompt is used for first list.

## MAP

Takes in comment batches and creates one argument list per batch.

## REDUCE

Takes in multiple argument lists and coalesces them into fewer lists (or one final list) using a reduction prompt.

## GROUND

Takes in argument lists and comment lists, refining arguments based on the comments. This operation doesn't have a visual explanation.
